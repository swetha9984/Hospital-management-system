Hospital-management-system

A Python console application to manage doctor and patient appointments. Users can schedule, confirm, and delete appointments, check doctor availability, and send reminders to both doctors and patients. It demonstrates OOP concepts like classes, inheritance, and methods while automating basic clinic workflows.

Features 
* Add multiple patients for a single appointment
* Check doctor availability before scheduling
* Confirm or delete appointments
* Send reminders to doctor and patients
* Display doctor and patient details

Future Enhancements
* Support multiple doctors and appointment slots
* Store appointments in a database or file
* Build a GUI or web interface
* Automated email notifications
